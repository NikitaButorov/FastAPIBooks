1.Скачать Pycharm/Visual Studio Code
2.Скачать XAMPP
3.Открыть папку ButorovSPTV21MVCBooks в Pycharm, VSCode
4.Запустить Apache, Mysql в XAMPP
5.Импортировать books.sql в MySQL
6.Открыть терминал Pycharm/Visual Studio Code
7.Прописать в терминале команду pip install -r requirements.txt
8.Запустить программу
9.Перейти по адресу http://localhost:3000/docs#
